<h1 align="center">
CONFORGE
</h1>
Generate config files and whatnot with a WebUI.

Built quickly with AI but honestly, its just to make my life easy.

## [IconRig](https://slothyacedia.github.io/conforge/IconRig.html)
Quickly select skill icons and get a export of the selected icons and a html snippet to paste into your GitHub READMEs.

## [ConfigParty](https://slothyacedia.github.io/conforge/ConfigParty.html)
A GUI that helps generate the .conf file that you can use to start your copyparty server.
